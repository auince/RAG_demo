# Ollama

所有 LangChain 与 [Ollama](https://ollama.com/) 的集成。

Ollama 允许您在本地运行开源模型（例如 [`gpt-oss`](https://ollama.com/library/gpt-oss)）。

有关支持的模型和变体的完整列表，请参阅 [Ollama 模型库](https://ollama.ai/library)。

## 模型接口

*   [ChatOllama](https://langchain-doc.cn/v1/python/integrations/chat/ollama): Ollama 聊天模型。
*   [OllamaLLM](https://langchain-doc.cn/v1/python/integrations/llms/ollama): (传统) Ollama 文本补全模型。
*   [OllamaEmbeddings](https://langchain-doc.cn/v1/python/integrations/text_embedding/ollama): Ollama 嵌入模型。
