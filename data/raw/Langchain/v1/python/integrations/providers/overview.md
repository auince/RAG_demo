# 集成包

LangChain Python 提供了一个广泛的生态系统，拥有 1000 多个集成，涵盖聊天和嵌入模型、工具和工具包、文档加载器、向量存储等等。

*   [**聊天模型**](../chat.html)
*   [**嵌入模型**](https://langchain-doc.cn/v1/python/integrations/text_embedding)
*   [**工具和工具包**](https://langchain-doc.cn/v1/python/integrations/tools)

要查看按组件类型划分的完整集成列表，请参阅侧边栏中的类别。
